Ubuntu Server

Selamat datang di sini menyediakan tutorial membuat ubuntu server.

Login

Login menggunakan username dan passwowrd yang kalian buat.

Login Sebagai Root

Akses root dengan perintah : sudo su , dan masukan password kembali mengunakan pasword login awal tadi.

Konfigurasi Jaringan

Cek IP dengan perintah : ifconfig. Jika terdapat IP seperti gambar diatas, ketikan perintah : nano /etc/resolv.conf dan atur seperti gambar diatas.

Bind 9

Instalasi Bind 9

Ketikan perintah : apt-get install bind9 (jika gagal ketik "sudo dpkg --configure -a"/"apt-get update")

Konfigurasi Default Zones

Ketikan perintah : nano /etc/bind/named.conf.default-zones . Dan sesuaikan seperti gambar diatas (Untuk IP di balik misalnya 192.168.10.1 menjadi 10.168.192, dan domain menggunakan domain sendiri)

Konfigurasi Local LoopBack Interface

Ketikan perintah : cp /etc/bind/db.local (spasi) /etc/bind/smk.db .Setelah itu ketikan : nano /etc/bind/smk.db , dan ikuti seperti gambar diatas dan sesuaikan IP dan Domain anda.

Konfigurasi Local LoopBack Interface 2

Ketikan perintah : cp /etc/bind/smk.db (spasi) /etc/bind/smk.ip .Setelah itu ketikan : nano /etc/bind/smk.ip (Sesuaikan IP dan Domain anda seperti gambar diatas, Angka 117 itu didapat dari Ip akhir)

Restart Bind9

Ketikan perintah : systemctl restart bind9

Testing DNS Server

Ketikan perintah : nslookup (Domain) , ketikan perintah : nslookup (IP) .Jika berhasil maka akan seperti gambar diatas.

Apache

Instalasi Apache

Ketikan perintah : apt-get install apache2 mysql-server phpmyadmin (isikan password untuk database mysql dan pilih apache)

Setting Apache

ketikan perintah : nano /etc/apache2/sites-available/000-default.conf (sesuaikan IP dan domain seperti digambar)

Restart Apache

Ketikan perintah : systemctl restart apache2

Web Server

Akses Web Server

Jika domain tidak bisa dibuka, ubah dns pada network connections menjadi IP server anda/IP virtual box

Akses PHPMYADMIN

Jika tidak bisa buka phpmyadmin, ketikan perintah : sudo -H nano /etc/apache2/apache2.conf (tambahkan include /etc/phpmyadmin/apache.conf pada baris paling bawah)

Restart Apache

Ketikan perintah : systemctl restart apache2
